# AI Data Analysis Workflows with Claude

> **Track:** Anthropic Enterprise Implementation (Track 1)
> **Course 3 of 6**

Build scalable pipelines that ingest documents, extract insights, and surface patterns using Claude as the analysis engine.

---

## Modules

### Module 1: Data Ingestion & Document Processing

**Learning Objectives:**
- Parse PDFs, CSVs, and unstructured text
- Chunk documents for context windows
- Build batch ingestion pipelines

### Module 2: Summarization at Scale

**Learning Objectives:**
- Implement map-reduce summarization
- Handle multi-document synthesis
- Control summary length and format

### Module 3: Pattern Recognition & Anomaly Detection

**Learning Objectives:**
- Prompt Claude for statistical reasoning
- Identify trends in tabular data
- Flag outliers with natural language explanations

### Module 4: Building Analysis Pipelines

**Learning Objectives:**
- Orchestrate multi-step analysis workflows
- Cache intermediate results
- Schedule recurring analysis jobs

### Module 5: Reporting & Visualization Integration

**Learning Objectives:**
- Generate executive summary narratives
- Feed Claude outputs into dashboards
- Automate report generation
